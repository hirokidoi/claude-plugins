# doi - Claude Code Plugin Marketplace

Claude Code プラグインのマーケットプレイスです。

## インストール

Claude Code 内で以下を実行:

```
/plugin marketplace add hirokidoi/claude-plugins
```

登録後、個別のプラグインをインストールできます。

## プラグイン一覧

| プラグイン | 説明 | インストール |
|---|---|---|
| [checklist-gate](plugins/checklist-gate/) | 操作前の明示的な確認（ack）を強制するゲートプラグイン | `/plugin install checklist-gate@doi` |

各プラグインの詳細は、リンク先の README を参照してください。
